# BaseUE5CPP 
BaseUE5CPP - проект созданный для изучения C++ языка в рамках движка Unreal Engine 5. 

В этом проекте создаются и просматриваются базовые конструкции C++ в Unreal Engine 5. 
- Создание управляемого персонажа
- Создание врага преследующего тебя
- Создание коспонентов: спавн врага, компонента жизней
- Создание интерфейсов для взаимодействия классов между собой, включая подбор предметов.

Также создание архитектуры проекта в которой можно регулировать через DataTable параметры и поля класса.
